# Multiply strings

[Problem link](https://leetcode.com/problems/multiply-strings)

## Solutions


### Solution.py
```py
# https://leetcode.com/problems/multiply-strings

class Solution:
    def multiply(self, num1: str, num2: str) -> str:
        return str(int(num1)*int(num2))
```
