# Add strings

[Problem link](https://leetcode.com/problems/add-strings)

## Solutions


### Solution.py
```py
# https://leetcode.com/problems/add-strings

class Solution:
    def addStrings(self, num1: str, num2: str) -> str:
        return str(int(num1)+int(num2))
```
