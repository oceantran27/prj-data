给定一个**32位**正整数 **n**，你需要找到最小的**32位**整数，其与 n 中存在的位数完全相同，并且其值大于n。如果不存在这样的**32位整数**，则返回-1。

**示例 1:**
```
输入: 12
输出: 21
```
**示例 2:**
```
输入: 21
输出: -1
```
