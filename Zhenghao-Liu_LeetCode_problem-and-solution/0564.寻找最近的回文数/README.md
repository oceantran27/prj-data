给定一个整数 n ，你需要找到与它最近的回文数（不包括自身）。

“最近的”定义为两个**整数差的绝对值**最小。

**示例 1:**
```
输入: "123"
输出: "121"
```
**注意:**

1. **n** 是由字符串表示的正整数，其长度不超过18。
2. 如果有多个结果，返回最小的那个。
