给定一个 N 叉树，找到其最大深度。

最大深度是指从根节点到最远叶子节点的最长路径上的节点总数。

例如，给定一个 ```3叉树``` :
 
![image](https://github.com/Zhenghao-Liu/LeetCode_problem-and-solution/blob/master/0559.N叉树的最大深度/narytreeexample.png)

我们应返回其最大深度，3。

**说明:**

1. 树的深度不会超过 ```1000```。
2. 树的节点总不会超过 ```5000```。
